## ACCF — Automated Control Cabinet Factory

Three machines, each an independent embedded system, coordinated over a two-layer communication stack: CAN-Bus internally, MQTT between stations and upstream.

| Station | Status |
|---|---|
| [Klemmen-Kommissionierstation](https://github.com/FloBraendle/accf-klemmen) | In development |
| [Litzenbearbeitungszentrum](https://github.com/FloBraendle/accf-litzen) | In development |
| [Verdrahtungsroboter](https://github.com/FloBraendle/accf-verdrahtung) | Concept phase |
